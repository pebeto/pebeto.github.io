# Jose Esparza — Personal Website

Personal portfolio site hosted on GitHub Pages.

## Stack

- **Next.js** 15 (App Router)
- **Tailwind CSS** 4
- **JuliaMono** font

## Local Development

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Build

```bash
npm run build
```

Exports a static site under `out/`.
